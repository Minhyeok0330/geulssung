# Task ID: 3
# Title: User Authentication System
# Status: pending
# Dependencies: 1, 2
# Priority: high
# Description: Implement user registration, login, and authentication system with email and nickname-based accounts.
# Details:
1. Create User model with Sequelize/TypeORM/Prisma
2. Implement registration endpoint:
   - Validate email format and uniqueness
   - Validate nickname uniqueness
   - Hash password securely (bcrypt)
   - Store user in database

3. Implement login endpoint:
   - Verify credentials
   - Generate JWT token
   - Return user info and token

4. Create authentication middleware:
   - Verify JWT token
   - Attach user to request object

5. Implement logout functionality
6. Add password reset flow (optional for MVP)

API Endpoints:
- POST /api/auth/register
- POST /api/auth/login
- POST /api/auth/logout
- GET /api/auth/me

# Test Strategy:
1. Test registration with valid and invalid inputs
2. Verify email and nickname uniqueness constraints
3. Test login with correct and incorrect credentials
4. Validate JWT token generation and verification
5. Test authentication middleware
6. Verify password hashing security
