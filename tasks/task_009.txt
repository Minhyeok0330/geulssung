3. Create project structure:
```

  /components
    /common
    /layout
    /characters
  /contexts
  /hooks
  /pages
  /services
  /utils
  App.js
  index.js
```

4. Set up routing
5. Configure API client
6. Implement authentication context/store
7. Create basic layout components
